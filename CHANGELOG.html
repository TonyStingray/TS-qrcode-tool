更新日誌 (Changelog)

v1.0.0 — 2026-05-28

首次公開發佈

功能

  • 支援 4 種資料類型：網址 (URL)、純文字、Wi-Fi、電子郵件
  • 即時預覽，300ms debounce 最佳化
  • 6 種點陣樣式 + 3 種外框樣式 + 2 種內框樣式
  • 自訂前景色、背景色、外框色、內框色
  • 可嵌入自訂 Logo（支援 PNG/JPG/GIF/SVG/WebP）
  • 匯出 PNG 或 SVG 格式，支援自訂檔名
  • 容錯等級選擇：L / M / Q / H
  • 3 欄桌面版 / 單欄手機版響應式佈局

安全

  • Content Security Policy (CSP) 嚴格設定
  • 零外部 CDN 依賴，所有資源本地託管
  • XSS 防護：使用 createElement + textContent
  • Logo 上傳深度驗證：MIME + 大小 + magic bytes + SVG 腳本掃描
  • 下載檔名清洗，防止路徑穿越
  • 隱私保護聲明與免責條款

效能 & 基建

  • PWA 支援：Service Worker + manifest.json，可離線使用
  • 完整 SEO 標籤：Open Graph、Twitter Card、meta description
  • 無障礙 (a11y)：aria-label、鍵盤導航、skip-to-content、reduced-motion
  • Tailwind CSS production build（minified 16KB）
  • GitHub Actions CI/CD 自動部署 + Lighthouse 審計
  • robots.txt + sitemap.xml + 404 頁面
  • 可插拔隱私友善分析模組（預設關閉）

版號遵循 語意化版本控制 (SemVer) 規範。
